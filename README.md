This is a simple raw and stupid the FE of [Tic Tac Toe](https://github.com/danielebarbaro/tic-tac-toe-api)

```bash
nvm use 20
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and Play. Have fun.
